# Nature-Style Manuscript Review Rules

Use these defaults unless the user provides a different target journal.

## Structure

- Do not keep a table of contents in the manuscript body. If needed, suggest moving it
  to planning material or a cover letter.
- Conclusions should synthesize the work and avoid new figure or table citations.
- Author contributions should name specific people and concrete responsibilities rather
  than using only generic statements.

## Clarity

- Define abbreviations, variables, and symbols before or at first use.
- Explain model names, metric names, feature labels, and mathematical notation in a way
  a reviewer outside the immediate project can follow.
- Long paragraphs should be split when they mix setup, result, interpretation, and caveat.

## Figures

- Captions for multi-panel figures must explain each panel label (`a`, `b`, `c`, `d`, etc.).
- Captions should identify what is plotted or shown before discussing the implication.
- Supplementary figures need enough context to stand alone.

## Formatting And Notation

- Normalize malformed Angstrom notation such as `A˚` to `Å` when the meaning is clear.
- Normalize scientific notation such as `8.5e-05` to `8.5 x 10^-5` or journal-preferred form.
- Keep variable styling, casing, spacing around operators, and metric names consistent.
